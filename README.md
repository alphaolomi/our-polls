Yarn is preferred

### Installation

Via Git

- Clone repo

```bash
git clone https://github.com/alphaolomi/vote-mono.git

cd wazo

yarn install
```

- Edit `.env` and set your database connection details

```bash
cp .env.example .env
```

- Development

```bash
yarn start

# debug mode
npm run debug
```

#### Production

```bash
yarn start
```
